API Assignment III - Select the Language

Assignment Description

  • Title:
    • Should be “Select the Language”
  • Instructions:
    • Should say “Determine the language of the code.”
  • Undo, Redo, Reset, Display Correct Answer Buttons:
    • Undo,Redo should undo and redo changes to font size. You do not have to undo/redo hiding.
    • Reset should reset the size of the answers and make them all visible.
    • “New” button should not be disabled
    • Display Correct Answer should be disabled
  • Code Well:
    • Should load code seen in the video from ex.data.json. Note that the for the second line, print(x + 7), the 7 should be dynamic.
  • Header:
    • Should say "Select Language:"
    • Should be centered between the code well and the right edge
  • Answers:
    • Paragraph element with an image element next to it
    • Image is supplied in the starter code.
    • Clicking an answer should make it italic and record the label as the user's answer.
    • Only one answer can be italic at a time.
    • Clicking the x image should hide the answer
    • Should be centered between the code well and the right edge
    • Hiding the answer should use the fade transition
  • Plus and Minus Buttons:
    • Clicking minus should make the answer font size smaller
    • Clicking plus should make the answer font size larger
    • The plus button should be 10px from the right edge and 10px from the top
    • The minus button should be 10px from the top and 10px from the plus button
    • Both buttons should have empty text, but should be keybound to - and +. (Binding to - and = is acceptable)
  • Submit:
    • Grades based on the which answer the user clicked. Python3 is correct.
    • Brings up the feedback modal display either “Correct!” or “Incorrect!”
    • Disables the plus and minus buttons
    • Disables clicking answers
  • State:
    • Font size of answers
    • Which answer was selected
    • Which answers were hidden
    • Whether the exercise was submitted
  • User interface should roughly match the video. No positioning may be hard-coded.
